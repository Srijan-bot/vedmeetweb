# VedaLife - Ayurvedic E-Commerce Website

A premium, dynamic e-commerce website for Ayurvedic medicines built with React, Tailwind CSS, and Framer Motion.

## Features

- **Premium Design**: Earthy color palette and elegant typography.
- **Dynamic Animations**: Smooth transitions using Framer Motion.
- **Product Catalog**: Filterable shop page with categories (Wellness, Skincare, Haircare, Immunity).
- **Responsive**: Fully optimized for mobile and desktop.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

1. Clone the repository (if applicable) or navigate to the project directory.
2. Install dependencies:

```bash
npm install
```

### Running the Project

To start the development server:

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173` (or the URL shown in the terminal).

### Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory.
